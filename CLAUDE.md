# Tài liệu Hỗ trợ Claude - Dự án antibrowsers

File này cung cấp bản đồ tài liệu và các lệnh vận hành nhanh cho dự án **antibrowsers**.

## 🗺️ Bản đồ tài liệu chính (Documentation Map)

Để hiểu sâu về cấu trúc hệ thống, APIs, tích hợp trình duyệt stealth và luồng lưu trữ dữ liệu, hãy truy cập trực tiếp file mục lục trung tâm:

👉 **[docs/INDEX.md](docs/INDEX.md)**

INDEX.md chứa danh sách liên kết và mô tả ngắn gọn về toàn bộ các tài liệu kỹ thuật chi tiết của hệ thống.

---

## 🛠️ Lệnh vận hành nhanh (Quick Commands)

* **Cài đặt toàn bộ dự án**: `npm install`
* **Chạy tích hợp dev server (Tauri + Sidecar + Vite)**: `npm run dev`
* **Đóng gói ứng dụng (Production Build)**: `npm run build`
